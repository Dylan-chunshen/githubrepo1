select-suborg

简介

select-suborg主要用于以下拉框的形式展示组织机构。包括普通下拉框和树形下拉框两种。

依赖关系

1、gov-util

目前只支持wegov平台（默认）。需要在\js-library\easyui-1.2.1\gov\config.js中配置oup_service实现类。也可以根据其他平台编写adapter类，但是该类必须实现gov.adapter.oup.OupService接口

各类平台的实现类如下：

平台 接口实现类
WEGOV gov.adapter.oup.wegov.WegovService

2、esayui

当控件为普通下拉框时，本标签调用的是easyui中的combobox控件；当控件为下拉树时，本标签调用的是easyui中的combotree控件。

使用方法

1、普通下拉框(不设tree属性或tree="false")

代码：


<select id="s2" name="s2" class="select-suborg" required="true" orgId="67072" ></select>

2、树形下拉框(tree="true")

代码：


<select id="s2" name="s2" class="select-suborg" tree="true" required="true" orgId="67072" ></select>

属性

普通下拉框请参考easyui中的combobox；树形下拉框请参考easyui中的combotree。以下只列出扩展后的属性：
属性 参考值 说明
orgId   必填属性。组织机构的根节点id，即使用本标签会将orgId对应的组织机构显示出现。
tree true或false 非必填属性。当tree为true时，使用树形控件，懒加载。
freeInput true或false 非必填属性。当freeInput为true时，允许用户可以不从选项中选取而是随意输入值提交。（目前只支持普通下拉框）
defaultValue   非必填属性。下拉框默认值，填组织机构id
defaultText   非必填属性。下拉框中的默认机构名称，和defaultValue对应。如果只填defaultValue，标签会自动从后台查询对应的Text。如果两个属性都设置，那么标签就不再从后台读defaultValue对应的text值。
readOnly   非必填属性。
textName   非必填属性。如果设置了textName，那么该下拉框的name会自动提交以此属性值为名称传入request中
