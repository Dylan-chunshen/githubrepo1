select-wad-code

简介

select-wad-code主要用于支持wad字典，提供普通下拉框和树形下拉框两种方式的控件。

依赖关系

1、wad

wad提供字典管理，支持wegov、ssm及CuteFramework等各种平台的字典接入。只需要在wad_config.properties中配置wad.dic.wadDicInstance属性。

各类平台的wad字典接口实现类如下：

平台 接口实现类
SSM wad2.ssm.DicHelper
WEGOV wegov.component.pub.service.CodeHelper

2、esayui

当控件为普通下拉框时，本标签调用的是easyui中的combobox控件；当控件为下拉树时，本标签调用的是easyui中的combotree控件。

使用方法

1、普通下拉框(不设tree属性或tree="false")

代码：


<select id="s1" name="s1" class="select-wad-code" name="select" required="true" tcode="SRC_PLATFORM" ></select>

2、树形下拉框(tree="true")

代码：


<select id="s1" name="s1" class="select-wad-code" name="select" required="true" tree="true" tcode="SRC_PLATFORM" ></select>

属性

普通下拉框请参考easyui中的combobox；树形下拉框请参考easyui中的combotree。以下只列出扩展后的属性：
属性 参考值 说明
tcode   必填属性。字典根code，即使用本标签会将tcode对应的所有字典显示出现。
tree true或false 非必填值。当tree为true时，使用树形字典，懒加载。
freeInput true或false 非必填属性。当freeInput为true时，允许用户可以不从选项中选取而是随意输入值提交。（目前只支持普通下拉框）
defaultValue   非必填属性。下拉框默认值，填字典的code
defaultText   非必填属性。下拉框中的默认字典的中文名称，和defaultValue对应。如果只填defaultValue，标签会自动从后台查询对应的Text。如果两个属性都设置，那么标签就不再从后台读defaultValue对应的text值。
readOnly   非必填属性。
textName   非必填属性。如果设置了textName，那么该下拉框的name会自动提交以此属性值为名称传入request中
